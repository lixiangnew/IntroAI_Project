# IntroAI_Project
人工智能导论课程项目作业

## 后端

### 1.创建虚拟环境
```bash
cd backend
python -m venv venv
```
### 2.激活虚拟环境
```bash
venv\Scripts\activate
```

### 3.安装依赖
```bash
pip install -r requirements.txt
```

### 4.运行服务器
```bash
python app.py
```

## 前端

### 1.下载nvm
前往[nvm-windows](https://github.com/coreybutler/nvm-windows/releases)下载

### 2.安装Node.js
```bash
nvm install 20
nvm use 20
```

### 3.安装依赖
```bash
cd frontend
npm install
```

### 4.运行服务器
```bash
npm run dev
```
